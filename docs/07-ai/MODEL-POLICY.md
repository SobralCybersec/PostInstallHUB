# Model and provider policy

**N/A.** PostInstallHUB uses no AI models and has no AI provider relationship in the product. There are no approved models, no fallback chains, no context limits to configure, and no data restrictions specific to model providers — because no data is sent to any model at runtime.

For AI tools used during *development*, see `AGENTS.md` in the repo root.
